首次配置指引（3 步，约 3 分钟）

第 1 步：注册 GitHub 账号
打开 github.com/signup 注册免费账号，记住用户名填到上面。

第 2 步：新建仓库放作品
打开 github.com/new：
· 名字填如 MEIJIA
· 选 Public 公开
· 勾 Add a README file
· Create repository。分支名一般是 main。

第 3 步：生成访问令牌
打开 github.com/settings/tokens/new：
· Expiration 选 No expiration
· 勾 repo 整项
· Generate token，复制 ghp_... 填到上面。

填完点「保存设置」→「测试连接」成功即可开始用。
